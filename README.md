# [cite_start]Institute of Digital Risk (IDR) - Brand & Homepage [cite: 1]

[cite_start]A responsive, single-page website and custom brand identity developed for the Institute of Digital Risk (IDR), an industry-led training and deployment institute[cite: 1]. 

## Overview
[cite_start]This project includes a complete logo design and a fully responsive frontend build, adhering to strict minimalist tech aesthetics and semantic web standards[cite: 6, 9, 45]. 

## Features
* [cite_start]**Semantic Web Structure:** Built entirely with semantic HTML5 tags (`<header>`, `<main>`, `<section>`, `<footer>`)[cite: 45].
* [cite_start]**Responsive Layout:** Utilizes vanilla CSS Flexbox and Grid to ensure a seamless experience across desktop and mobile devices[cite: 46].
* [cite_start]**Framework-Free:** Developed using strictly vanilla HTML, CSS, and JavaScript—no Bootstrap or external CSS libraries were used[cite: 51].
* [cite_start]**Interactive UI:** Features a sticky navigation bar, smooth scrolling to sections, and basic hover states for all interactive elements[cite: 48, 50].
* [cite_start]**Custom Brand Identity:** Includes a geometric, cube-inspired logo in both icon-only and text-variant formats (SVG and PNG), utilizing a strict orange, white, and black color palette[cite: 7, 9, 10, 13].

## Tech Stack
* HTML5
* CSS3
* JavaScript (ES6+)
* Figma (Logo & Asset Design)
