# Institute of Digital Risk (IDR) - Brand & Homepage

A responsive, single-page website and custom brand identity were developed for the Institute of Digital Risk (IDR), an industry-led training and deployment institute. 

## Overview
This project includes a complete logo design and a fully responsive frontend build, adhering to strict minimalist tech aesthetics and semantic web standards. 

## Features
**Semantic Web Structure:** Built entirely with semantic HTML5 tags (`<header>`, `<main>`, `<section>`, `<footer>`).
**Responsive Layout:** Utilizes vanilla CSS Flexbox and Grid to ensure a seamless experience across desktop and mobile devices.
**Framework-Free:** Developed using strictly vanilla HTML, CSS, and JavaScript—no Bootstrap or external CSS libraries were used.
**Interactive UI:** Features a sticky navigation bar, smooth scrolling to sections, and basic hover states for all interactive elements.
**Custom Brand Identity:** Includes a geometric, cube-inspired logo in both icon-only and text-variant formats (SVG and PNG), utilizing a strict orange, white, and black color palette.

## Tech Stack
* HTML5
* CSS3
* JavaScript (ES6+)
* Figma (Logo & Asset Design)
